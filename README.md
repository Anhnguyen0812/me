# Nguyen Phi Anh - Developer Portfolio ⌨️

> Neo-Brutalism style portfolio website built with vanilla HTML, CSS & JavaScript.

## 🚀 Live Demo

Visit: [https://anhnp.github.io](https://anhnp.github.io) *(update with your URL)*

## 🛠 Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Neo-Brutalism design system (no frameworks)
- **JavaScript** - Vanilla JS for interactions
- **Google Fonts** - Space Grotesk + JetBrains Mono
- **Material Symbols** - Icon system

## 📁 Structure

```
me/
├── index.html          # Main page
├── css/
│   └── style.css       # Complete design system
├── js/
│   └── main.js         # Interactions & animations
└── README.md
```

## 🎨 Design

- **Style**: Neo-Brutalism + Chunky 3D
- **Features**: 3D keycap skill tags, retro window project cards, terminal contact section
- **Responsive**: Desktop, tablet & mobile with bottom navigation

## 📦 Deploy to GitHub Pages

1. Create a new GitHub repository
2. Push this code to `main` branch
3. Go to Settings → Pages → Source: `main` / `root`
4. Your site will be live at `https://username.github.io/repo-name`
